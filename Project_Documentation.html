1. Project Overview

DLPS Enterprise is a comprehensive security solution designed to prevent the unauthorized transmission of sensitive data. The system acts as a protective shield, analyzing files and data in real-time to detect Identifiable Information (PII), financial records, and confidential corporate documents before they can leave the secure environment.

4. Team Work Distribution (4 Members)

The project development is divided into four distinct modules to ensure efficiency and specialization.

Member 1: System Architect & Backend Core
  • Database Design: Designing the schema.sql (Users, Policies, Logs).
  • Authentication Module: Login, Registration, Forgot Password, Session Management.
  • API Development: Creating RESTful endpoints for frontend communication.
  • Security Hardening: Implementation of auth middleware and encryption.
Member 2: Security Engine & Algorithms
  • Scanning Logic: Developing the regular expressions and pattern matching algorithms.
  • File Processing: Handling file uploads, reading content, and parsing data.
  • Policy Enforcement: Logic to check files against active Database Policies.
  • Risk Assessment: Calculating risk levels (Low, Medium, Critical) based on findings.
Member 3: Admin Portal & Analytics
  • Admin Dashboard: Creating the main analytics hub with Chart.js visualizations.
