Q1: How do you secure user passwords in the database?
A: We use the password_hash() function with the Bcrypt algorithm, which automatically handles salting. We never store plain-text passwords.
Q2: How does your session management prevent Session Hijacking?
A: We regenerate session IDs on login using session_regenerate_id() and enforce custom expiry times in the database to prevent stale sessions from being reused.

üéì Professor's Questions

Q1: How does the system detect an SSN? Explain the pattern.
A: We use a Regex pattern like \b\d{3}-\d{2}-\d{4}\b. The \b ensures word boundaries so we don't accidentally flag part of a longer number.
Q2: What happens if a file is too large?
A: We check $_FILES['file']['size'] before processing. If it exceeds the limit (e.g., 5MB), we reject it immediately to prevent Denial of Service (DoS) attacks.

Q1: How did you implement the Dark Mode theme?

Q2: How does the "Forgot Password" page communicate with the user?
